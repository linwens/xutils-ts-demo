## 介绍
----

`xutils-ts` 是一款基于 `typescript` 开发的的js工具库。

## 特性
----

- 基于 `typescript` 的组件开发
- 借鉴（抄袭）`lodash` 源码
- 项目发包基于 `travis` 集成，基于 `coveralls` 做测试覆盖率检测
